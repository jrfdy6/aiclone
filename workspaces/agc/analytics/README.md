# AGC Analytics

Store workspace-local traction artifacts here.

First tracked traction signal:
- qualified `email communication from prospective buyers`

Use this folder for:
- inbound email evidence
- opportunity qualification notes
- positioning-response learnings
- any buyer signal that explains why the next capture move is justified

Canonical files:
- `agency_prime_watchlist.csv` for the ranked agency and prime target map
- `opportunity_ledger.csv` for stage, score, next move, and evidence path
- `inbound_email_log.csv` for qualified email intake and follow-up status

Do not mix in other workspace analytics.
