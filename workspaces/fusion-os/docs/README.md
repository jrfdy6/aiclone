# Fusion OS Knowledge Network

This folder is a workspace-local markdown network for Fusion research, messaging, and content craft.

It is not canon.
It does not replace the root workspace identity.
It exists so the Fusion lane can think with more depth without bloating `IDENTITY.md` or leaking assumptions into other workspaces.

When the task is about the recurring PM -> packet -> execution -> write-back loop, start with [`execution_lane.md`](/Users/neo/.openclaw/workspace/workspaces/fusion-os/docs/execution_lane.md) before drilling into research or messaging docs.

## What This Folder Is For

- Turn research into usable operating context
- Support posts, captions, landing copy, email copy, image direction, and hashtag choices
- Keep the voice human, calm, and family-safe
- Separate durable structural truths from unverified promotional claims

## Core Operating Truth

The Fusion content lane should not sound like a brand trying to win an argument.
It should sound like a steady human voice that understands what school stress does to students and families.

## Hard Rules

- Do not treat every NotebookLM summary as equally true.
- Do not move research conclusions into root identity files unless explicitly requested.
- Do not use exact stats in public-facing copy unless they are current and verified.
- Do not write in generic AI-marketing language.
- Do not center prestige when the audience is in pain.
