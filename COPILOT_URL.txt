📋 QUICK DEPLOY CHECKLIST
=====================================

1. Install firebase-admin:
   cd frontend && npm install firebase-admin

2. Set Railway env vars (frontend service):
   - NEXT_PUBLIC_API_URL
   - FIREBASE_SERVICE_ACCOUNT
   - DEFAULT_USER_ID
   - NEXT_PUBLIC_SITE_URL

3. Ingest optimized persona:
   curl -X POST https://your-backend.up.railway.app/api/ingest/upload \
     -F "file=@JOHNNIE_FIELDS_PERSONA_OPTIMIZED.md" \
     -F "user_id=default-user"

4. Update robots.txt with your actual URL

5. Deploy:
   git add . && git commit -m "Add Copilot KB" && git push

6. Test:
   - Visit: https://your-frontend.up.railway.app/kb
   - Disable JS and verify content loads
   - Check sitemap: /sitemap.xml

7. Add to Copilot and wait 24-48 hours
