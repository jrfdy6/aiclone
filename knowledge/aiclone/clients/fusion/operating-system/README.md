# Fusion OS Operating System

This folder translates a mature ed-tech operating structure into a Fusion-native operating system.

## Core Files

- `fusion-os-school-health-system.md`: the operating model
- `fusion-os-school-health-schema.json`: import-ready data shape
- `fusion-os-school-health-scorecard-template.csv`: assessment template
- `fusion-os-nuance-intake.md`: correction-first intake for Fusion-specific business nuance
- `fusion-os-outreach-admissions-prompt-library.md`: reusable prompts for outreach, admissions, referral strategy, and coaching
- `fusion-dc-admissions-playbook-v1.md`: recommended-default decision logic for interpreting Fusion DC outreach and admissions data
- `fusion-os-master-synthesis-from-221-screenshots.md`: the full canonical synthesis of the 221-file archive through the Fusion OS lens

## Design Goal

Do not mirror another organization's language blindly.

Instead, use the source material to preserve what matters:

- operating visibility
- stage-by-stage accountability
- pro portfolio ownership
- package mix visibility
- session and staffing visibility
- campus social confidence
- risk escalation
- intervention selection
- student-success and retention discipline
- team capability management
- systems and reporting hygiene

Then express it in Fusion OS as your own model.
